2D-CCPF Model

Problem

Consider the 2D-CCPF model with Dirichlet boundary conditions : given a vector function $R=(R^m,R^c)$ and $u_0\in H^1_0(\Omega_m)$, find a vector function $u = (u^m, u^c)$ such as : \begin{equation} \begin{cases} \begin{aligned} S\frac{\partial u^m}{\partial t}-\nabla\cdot\left(\mathbb K\nabla u^m\right) &= -\alpha_{ex} \left(u^m-u^c\right)\delta_{y} +R^m \text{ in } \Omega_m\\ -\frac{\partial}{\partial x}\left(D\frac{\partial u^c}{\partial x}\right) &= \alpha_{ex}\left( {u^m}_{|y=0} -u^c\right)+R^c \text{ in } \Omega_c\\ u^m&=0\text{ on } \partial \Omega_m\\ u^c&=0\text{ on } \partial\Omega_c\\ u^m(0,\cdot) &= u_0\text{ on } \Omega_m \end{aligned} \end{cases} \end{equation}

In this implementation, we set all the constant values at $1$.

Weak formulation

The weak formulation of semi-discretization is : given a function $u_0$ of $H^1_0(\Omega_m)$, find a sequence $(u_p)_{1\leq p\leq N}$ of $W$ such as : \begin{equation} \begin{cases} a_p(u_p,w) = \tau_p (R_p,w)+S(u^m_{p-1},w^m), \forall w\in W,\\ u^m_0 = u_0 \end{cases} \end{equation} where $a_p$ is a bi-linear form defined on $W\times W$ by : $a_p(v,w) = a^m_p(v,w)+a^c_p(v,w)$, with : \begin{equation} \begin{aligned} a^m_p(v,w) &= \int_{\Omega_m}S v^m w^m+\tau_p\int_{\Omega_m}\mathbb K \nabla v^m\cdot \nabla w^m+\tau_p\alpha_{ex}\int_0^L\left(v^m(x,0)-v^c(x)\right)w^m(x,0)\\ a^c_p(v,w) &=\tau_p\int_0^L D\frac{d v^c}{d x}\frac{dw^c}{dx} +\tau_p\alpha_{ex}\int_0^L \left(v^m(x,0) -v^c(x)\right) w^c(x)dx \end{aligned}. \end{equation}

Meshes generations

For each $p$, we define

  • the anistropic triangulation $\mathcal T_{ph}$ based on triangular, for porous media domain $\Omega_m$ as follows. First on the $y$-direction, we divide $\left[-\frac{1}{2}, \frac{1}{2}\right]$ into $m$ segments with $m+1$ points $y_i$, where : \begin{equation} \begin{aligned} y_i &= -\frac{1}{2} +\frac{\sin(\frac{i\pi}{m})}{2}, i = 0,1,\dots, \frac{m}{2}\\ y_i &=\frac{1}{2} -\frac{\cos{\left(\frac{i\pi}{m} -\frac{\pi}{2}\right)}}{2}, i = \frac{m}{2}+1, \dots, m \end{aligned} \end{equation}

Next, on $x$-direction, we divide $[0,1]$ into $n$ equal segments with $n+1$ points.

  • the triangulation $\mathcal P_{ph}$ is a restriction of $T_{ph}$ on conduit line $\Omega_c$

Approximation spaces

For a triangulation $\mathcal T_{ph}$ of $\Omega_m$ consisting of triangular, we approximate the hydraulic head in porous matrix $\Omega_m$ in the Crouzeix-Raviart element space and the hydraulic head in pipe conduit in the space of piecewise constant functions, namely :

\begin{equation} \begin{aligned} W^m_{ph} &=\left\{u^m_{ph}\in L^2(\Omega_m) : {u^m_{ph}}_{\big| T}\in \mathbb P^1(T), \forall T\in \mathcal T_{ph}, \int_E \left[u^m_{ph}\right] =0,\forall E\in \mathcal E\right\},\\ W^c_{ph} &= \left\{u^c_{ph}\in C^0(\Omega_c) : {u^c_{ph}}_{\big|E}\in \mathbb P^0(E), \forall E\in\mathcal P_{ph} \right\}. \end{aligned} \end{equation}

def shape_functions(nodes):
  """Compute the coefficients of shape functions of an triangle

  Returns:
    coefficients:
        coefficients[i, j] : j-th coefficient of the i-th local shape function
  Note :
      the shape function is given by : coefficients[i, 0] + coefficients[i, 1]*x_1 + coefficients[i, 2]*x_2
  """
  #[0,1], [0,2], [1,2]
  milieu = np.zeros((3, 2))
  milieu[0,:] = (nodes[0,:] + nodes[1,:]) / 2
  milieu[1,:] = (nodes[0,:] + nodes[2,:]) / 2
  milieu[2,:] = (nodes[1,:] + nodes[2,:]) / 2

  #Define a matrix using the
  A = np.hstack((np.ones((3, 1)), milieu))
  I = np.eye(3)
  X = np.linalg.solve(A, I) #Solve AX = I
  coefficients = X.T
  return coefficients

from math import sqrt
def quadrature_on_triangle(S):
  L = 7#Number of Gauss' points
  a1 = (6-sqrt(15))/21
  a2 = (6+sqrt(15))/21
  #This points is expressed in barycentric coordinates
  gauss_points = np.array([[1/3, 1/3, 1/3], [a1, a1, (1-2*a1)], [a1, (1-2*a1), a1],
   [(1-2*a1), a1, a1], [a2, a2, (1-2*a2)], [a2, (1-2*a2), a2], [(1-2*a2), a2, a2]])
  omega1 = S*(155-sqrt(15))/1200
  omega2 = S*(155+sqrt(15))/1200
  weights = np.array([9*S/40, omega1, omega1, omega1, omega2, omega2, omega2])
  return gauss_points, weights

  Returns a real value

    f: a function

  """

#@title Quadrature 1D
def quadrature_on_segment(a,b):
  m = (a+b)/2.0
  delta = b-a
  gauss_points = np.array([m+delta/2.0, m-delta/2.0, m])
  weights = np.array([delta/6.0, delta/6.0, 2*delta/3.0])
  return gauss_points, weights

#Perform 1D integrale of f on (a,b)
def integrale1D(a,b, f):
  """Returns a real value

  Input:
    a,b : two real such as a<b
    f : a 1 variable function

  Output : real number
  """
  gauss_points, weights = quadrature_on_segment(a,b)
  return f(gauss_points).dot(weights)

Assembling

Let's denote by $\left(\varphi_i ^m \right)_{0\leq i\leq N}$ of $W^m_{ph}$ and by $\left(\varphi^c_i\right)_{0\leq i\leq P}$ basis of $W^c_{ph}$. The full discretization is equivalent to : given an approximation $u_0h$ of $u_0$, find a sequence $\left(u_{ph}\right)$ such that : \begin{equation} \begin{aligned} % a^m_{ph} (u_{ph}, w_{ph}) &= \tau_p \left(R^m_p, w^m_{ph}\right) + S\left(u^m_{ph},w^m_{ph}\right), \forall w^m_{ph} \in W^m_{ph}\\ % a^c_{ph} (u_{ph}, w_{ph}) &= \tau_p \left(R^c_p, w^c_{ph}\right), \forall w &\sum_{T\in \mathcal T _{ph}} \left[S\int_T u^m_{ph}w^m_{ph} + \tau_p \mathbb K \int_T \nabla u^m_{ph}\cdot \nabla w^m_{ph}+\tau_p\alpha_{ex} \int_T (u^m_{ph}-u^c_{ph})w^m_{ph}\delta_y\right] = \sum_{T\in \mathcal T_{ph}} \left[\tau_p \int_T R^m_p w^m_{ph} + S\int_T u^m_{p-1,h}w^m_{ph}\right]\\ &\sum_{E\in \mathcal P_{ph}} \left[\tau_p \int_E D\frac{d u^c_{ph}}{dx} \frac{d w^c_{ph}}{dx} + \tau_p\alpha_{ex}\int_E{(u^m_{ph}}_{|y=0} - u^c_{ph})w^c_{ph} \right] = \tau_p \sum_{E\in \mathcal P_{ph}} \int_E R^c_p w^c_{ph} \end{aligned} \end{equation}

Let $U^m_p = (u^m_{p1}, \dots, u^m_{pN})$ the coordinates of $u^m_{ph}$ in the basis $\left(\varphi_i ^m \right)_{0\leq i\leq N}$ and $U^c_p = (u^c_{p1}, \dots, u^c_{pP})$ the coordinates of $u^c_{ph}$ in the basis $\left(\varphi^c_i\right)_{0\leq i\leq P}$. The previous system is equivalent to the following :

\begin{equation} \begin{aligned} \sum_{i=0}^N u^m_{pi}\underline A_{ji} -\sum_{k=0}^P u^c_{pk} \underline B_{jk} &= \underline E_j, 0\leq j\leq N\\ \sum_{i=0}^N u^m_{pi}\underline C_{li} -\sum_{k=0}^P u^c_{pk} \underline D_{lk} &= \underline F_j, 0\leq l\leq P\\ \end{aligned} \end{equation} where

\begin{equation} \begin{aligned} \underline A_{ij} &= \sum_{T\in \mathcal T _{ph}} \left[S\int_T \varphi^m_i\varphi^m_j + \tau_p \mathbb K \int_T \nabla \varphi^m_i\cdot \nabla \varphi^m_j+\tau_p\alpha_{ex} \int_T \varphi^m_i \varphi^m_j\delta_y\right]\\ \underline B_{kj} &= \tau_p \alpha_{ex} \sum_{T\in \mathcal T_{ph}} \int_T\varphi^c_k \varphi^m_j \delta_y\\ \underline C_{ij} &= \tau_p\alpha_{ex} \sum_{E\in\mathcal P_{ph}} \int_E {\varphi^m_i}_{|y=0} \varphi^c_j dx\\ \underline D_{kj} &= \tau_p D\sum_{E\in \mathcal P_{ph}} \int_E \frac{d\varphi^c_k}{dx}\frac{\varphi^c_j}{dx} -\tau_p\alpha{ex} \sum_{E\in \mathcal P_{ph}} \int_E\varphi^c_k\varphi^c_j\\ \underline E_j &= \sum_{T\in \mathcal T_{ph}} \left[\tau_p \int_T R^m_p \varphi^m_{j} + S\int_T u^m_{p-1,h}\varphi^m_{j}\right] \\ \underline F_j &= \tau_p \sum_{E\in \mathcal P_{ph}} \int_E R^c_p w^c_{ph} \end{aligned} \end{equation}

This leads to the linear system defined by :

\begin{equation} \begin{pmatrix} \underline A & -\underline B\\ \underline C & \underline D \end{pmatrix} \begin{pmatrix} U^m_p\\ U^c_p \end{pmatrix} = \begin{pmatrix} \underline E\\ \underline F \end{pmatrix} \end{equation}
